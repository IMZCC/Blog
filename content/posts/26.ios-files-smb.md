文件应用内复制文件到自建nas服务中报错，不能完成操作。（OSStatus 错误 100093）

## 编辑samba配置文件
```bash
vim /etc/samba/smb.conf
```

## global节点下增加配置
```conf
[global]
    vfs objects = acl_xattr catia fruit streams_xattr
```

## 重启smbd服务
```bash
systemctl restart smbd
```
